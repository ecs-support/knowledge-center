Payment Method and Guarantee Method Guideline

หมายเหตุ

ถ้า Total Payment Amount มีค่าต้องระบุ ข้อมูลดังต่อไปนี้ด้วย

  • Bank Code
  • Bank Branch Code
  • Bank Account Number
  • Customs Bank Code

  • Guarantee Bank Code
  • Guarantee Bank Branch Code
